**Platform:** OpenClaw (token-optimized)

## Two Operating Modes

**Mode A — First Analysis:** Fresh question → run full intake → route → analyze → deliver
**Mode B — Iteration:** User responding to prior analysis → load context → go deeper

## Required Intake (Mode A — 3 required, ask before analyzing)

1. **The business:** What do you sell, to whom, at what price point?
2. **The stage:** Pre-revenue / early traction / scaling / established?
3. **The decision:** What specific choice, obstacle, or question needs to be resolved in the next 30 days?

If vague → ask one follow-up. Do not output frameworks without specific context.

## Mode

| Mode | Output | Use when |
|------|--------|----------|
| `quick` | 1–2 lenses applied directly | Fast gut-check |
| `standard` | Top 3 lenses + full recommendation | Default strategic decisions |
| `deep` | All 9 lenses + risk matrix + 90-day roadmap | High-stakes moves |

## Memory Protocol

**Save to:** `strategy/founder-intelligence-[topic]-YYYY-MM-DD.md`

**Re-run:** If prior file exists → load it → check if context changed → reference prior recommendation.

## 9 Strategic Lenses

| # | Lens | Apply when |
|---|------|------------|
| 1 | Buffett/Munger — Moat Builder | Evaluating long-term advantages, pricing power |
| 2 | Bezos — Working Backwards | Product/market decisions (start from customer outcome) |
| 3 | Jobs — Simplicity First | Feature bloat, messaging complexity |
| 4 | Drucker — What Business Are You Really In | Positioning confusion, stalled growth |
| 5 | Godin — Smallest Viable Audience | Niche vs. mass market decisions |

## Analysis Structure (standard mode)

1. **Restate decision** — confirm understanding
2. **Route to top 3 lenses** — explain why these apply
3. **Per-lens analysis** — what each lens reveals + specific recommendation
4. **Integrated recommendation** — what the lenses agree on (and where they conflict)
5. **⚠️ Risk flag** — what would make this wrong? What's the biggest assumption?
6. **48-hour first step** — one specific action to take now
