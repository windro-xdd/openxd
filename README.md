# OpenXD

AI-powered coding agent that lives in your terminal.

## Install

```bash
curl -fsSL windro-xdd.github.io/openxd/install | bash
```

### Specific version

```bash
curl -fsSL windro-xdd.github.io/openxd/install | bash -s -- --version 1.0.0
```

### From local binary

```bash
./install --binary /path/to/openxd
```

## Usage

```bash
cd your-project
openxd
```

First run walks you through setup — name, preferences, personality. Takes 30 seconds.

## What it does

- AI coding assistant in your terminal
- Persistent memory across sessions
- Browser automation
- Multi-model support
- First-run onboarding that learns your preferences

## Requirements

- macOS (Intel/Apple Silicon), Linux (x64/arm64), or Windows (x64)
- bash, curl, tar/unzip
